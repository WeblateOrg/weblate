TermBase eXchange format
------------------------

.. versionadded:: 4.5

TBX is an XML format for the exchange of terminology data.

.. seealso::

    `TBX on Wikipedia <https://en.wikipedia.org/wiki/TermBase_eXchange>`_,
    :doc:`tt:formats/tbx`,
    :ref:`glossary`
